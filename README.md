# my-dlna-server

```
usage: java -jar my-dlna-server-*-jar-with-dependencies.jar
       [-h] [-p PORT] [-H HOST] -d DIR

simple DLNA server for serving media files

named arguments:
  -h, --help             show this help message and exit
  -p PORT, --port PORT   http port (default: 8899)
  -H HOST, --host HOST   (default: 192.168.1.104)
  -d DIR, --dir DIR      media file dir

```
